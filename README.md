# Delax Mart Product Link Storage

A simple local website for storing product links with unique IDs and quick search.

## How to use

1. Open `index.html` in your browser.
2. Enter the product name, URL, select the store, and add optional notes.
3. Click `Generate ID` if you want a new unique ID.
4. Click `Save product` to store it.
5. Use the search box to find a stored product by ID, name, or store.

## Features

- Saves data in browser local storage
- Auto-generates unique IDs like `DLX-xxxxx`
- Search by ID
- Delete saved products

## Notes

- This site works without a server.
- Data is saved in the browser, so it stays until you clear storage or delete entries.
